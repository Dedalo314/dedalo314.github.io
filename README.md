# Aletheia: Frontiers of Thought

A blog focused on exploring the underlying principles of the universe and the intelligence that strives to understand it.

## Features

- **LaTeX Support:** Render mathematical formulas with ease.
- **Dynamic Animations:** Interactive visualizations for complex concepts.
- **Content Collections:** Well-organized posts and documentation.

## Tech Stack

- **Framework:** Astro
- **Styling:** Tailwind CSS
- **Testing:** Playwright
- **Math:** remark-math / rehype-katex
