# Math, Physics, and AI Blog

A personal blog built with [Astro](https://astro.build/) focused on mathematics, physics, and artificial intelligence models/agents.

## Features

- **LaTeX Support:** Render mathematical formulas with ease.
- **Dynamic Animations:** Interactive visualizations for complex concepts.
- **Content Collections:** Well-organized posts and documentation.

## Tech Stack

- **Framework:** Astro
- **Styling:** Tailwind CSS
- **Testing:** Playwright
- **Math:** remark-math / rehype-katex
